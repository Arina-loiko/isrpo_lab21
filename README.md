## Описание (что изучили)

В этой лабораторной работе изучили работу с DOM в JavaScript:
- поиск и создание элементов через document.getElementById и document.createElement
- обработку событий (click, keydown)
- работу с формами (получение значения input, очистка поля)
- динамическое обновление интерфейса без перезагрузки страницы
- использование setInterval для создания секундомера
